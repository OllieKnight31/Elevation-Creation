Endpoint POST https://webinar-system-five.vercel.app/api?source=meta
How to connect:
  1. Go to Meta Business Manager → Business Settings → System Users
  2. Create a System User or select an existing one
  3. Click Generate New Token — select ads_read permission
  4. Copy the access token and set it as META_ACCESS_TOKEN in Vercel env vars
  5. The tracker will automatically pull your ad campaign data

WebinarJam
Checking...
Endpoint POST https://webinar-system-five.vercel.app/api?source=webinarjam
How to connect:
  1. Log into WebinarJam → select your webinar
  2. Go to Configuration → Integrations tab
  3. Scroll to API / Webhook Notifications
  4. Paste the endpoint URL above as the Notification URL
  5. Select events: Registration, Attended, No-show, Left early
  6. Click Save
Open WebinarJam ↗

Calendly
Checking...
Endpoint POST https://webinar-system-five.vercel.app/api?source=calendly
How to connect:
  1. Log into Calendly → click Integrations & Apps
  2. Scroll to API & Webhooks section
  3. Click Webhook Subscriptions
  4. Click + New Webhook
  5. Paste the endpoint URL above
  6. Select events: invitee.created and invitee.canceled
  7. Set scope to your organisation, click Create
